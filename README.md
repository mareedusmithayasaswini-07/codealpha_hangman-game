# 🎯 Hangman Game using Python

A simple and interactive **console-based Hangman Game** developed using Python.
The player has to guess the hidden word one letter at a time within limited attempts.

---

## 🚀 Features

✨ Random word selection
✨ Difficulty levels (Easy, Medium, Hard)
✨ Hint system 💡
✨ Score tracking 🏆
✨ Play again option 🔁
✨ User-friendly console interaction

---

## 🛠️ Technologies Used

* Python
* Random Module

---

## 📌 Concepts Used

* Functions
* Loops
* Conditional Statements
* Lists
* Strings
* User Input Handling

---

## 🎮 How to Play

1️⃣ Run the Python program
2️⃣ Choose a difficulty level
3️⃣ Guess letters one by one
4️⃣ Try to find the complete word before attempts run out
5️⃣ Use hints if needed 💡

---

## ▶️ Difficulty Levels

| Level  | Attempts |
| ------ | -------- |
| Easy   | 8        |
| Medium | 6        |
| Hard   | 4        |

---

## 🏆 Game Features

* Tracks wins and losses
* Prevents repeated guesses
* Displays guessed letters
* Allows multiple rounds without restarting the program
